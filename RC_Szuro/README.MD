 # RC szűrő

Szül.: 2006.12.23.   

Aluláteresztő RC szűrő kapcsolás 

*Adatok:*  

$R = 12 Ohm $   
$C = 23 uF $

*Határfrekvencia:*   
   
$fh= {1 \over (2 * Pi * R * C)} $   
$fh= {1 \over (2 * Pi * 12R * 23uF)} $   
$fh= 576,6483 Hz $
